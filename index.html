Key Features

🎯

Adaptive Home Surfaces

Interface adapts to Reader, Student, or Seeker archetypes based on behavior

🧠

Memorization Tracker

Spaced-repetition scheduling with on-device voice verification via Whisper

🔥

Goals & Streaks

Reading plans, activity heatmaps, and streak tracking synced via User API

🔖

Cloud Bookmarks

Bookmarks and collections synced across all devices via Quran.Foundation

💬

Quran Reflect

Community reflections feed with verse-specific posts and social sharing

🔐

OAuth2 + PKCE

Secure authentication with offline refresh tokens and scoped API access
