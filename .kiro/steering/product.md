# Product Overview

**Hafiz** is a Flutter-based Quran memorization and reading app designed to help Muslims read, memorize, and verify their Quran recitation.

## Core Features

- **Quran Reading**: Full Quran text with multiple Mushaf types (Madani, Indo-Pak, Egyptian, Warsh)
- **Voice Verification**: Speech-to-text recitation checking with error tracking
- **Audio Playback**: Background audio with verse highlighting and controls
- **Bookmarks**: Save and manage reading positions (page and verse bookmarks)
- **Search**: Find verses across the entire Quran
- **Deep Linking**: Share and navigate to specific verses or pages
- **Offline-First**: Quran text bundled locally, works without internet
- **Juz Navigation**: Quick access to any of the 30 Juz
- **Practice List**: Mark difficult verses for focused review (Tathbeet)
- **Hifz Mode**: Blur verses to test memorization

## Target Platforms

- Android (primary, published on Google Play)
- iOS
- Web, Linux, macOS, Windows (supported)

## Key Principles

- **Quran Text Integrity**: Arabic text is bundled locally from verified Tanzil source (CC BY-ND 3.0) to prevent tampering
- **Offline Capability**: Core reading functionality works without network
- **Accessibility**: Supports Arabic (RTL) and English localization
- **Non-Profit**: Built as a good deed, no monetization

## User Journey

1. Onboarding → Select Mushaf type (Madani, Indo-Pak, Egyptian, or Warsh)
2. Home screen → Browse Surahs, Juz, or continue reading
3. Surah/Mushaf view → Read with verse-by-verse display
4. Voice verification → Record and check recitation accuracy
5. Audio player → Listen to professional recitations
6. Practice mode → Review marked verses and track progress

## Mushaf Types

### Madani (Uthmani) - 604 pages
- Most common in Saudi Arabia and Middle East
- Standard Uthmani script
- Default selection

### Indo-Pak - 558 pages
- Common in India, Pakistan, Bangladesh
- Distinctive Nastaliq-style script
- Different page breaks from Madani

### Egyptian (Standard) - 604 pages
- Used in Egypt and some African countries
- Similar to Madani with slight variations

### Warsh (North African) - 604 pages
- Common in Morocco, Algeria, Tunisia
- Different recitation style (Qiraat)
- Unique orthography
