## Task 2: Uthmani Rasm Data Migration

### 2a. `scripts/download_uthmani.dart` — CREATE (one-shot CLI)
```
dart run scripts/download_uthmani.dart
```
- Hits `GET https://api.quran.com/api/v4/quran/verses/uthmani?chapter_number={N}` for N=1..114
- Parses response `{"verses": [{"verse_key": "N:V", "text_uthmani": "..."}]}`
- Writes to `assets/quran/uthmani/surah_{N}.json` matching existing schema:
  ```json
  {"chapter": [{"chapter": N, "verse": V, "text": "..."}]}
  ```
- Run locally, commit the regenerated 114 files

### 2b. KFGQPC Uthmani Taha Naskh Font — DOWNLOAD + ADD
- Download from KFGQPC official source
- Place at `assets/fonts/KFGQPC_Uthmani_Taha_Naskh.ttf`
- Add to `pubspec.yaml` fonts section:
  ```yaml
  - family: UthmaniTahaNaskh
    fonts:
      - asset: assets/fonts/KFGQPC_Uthmani_Taha_Naskh.ttf
  ```

### 2c. TextStyle updates — EDIT existing files
Update `fontFamily` from `'Amiri'` to `'UthmaniTahaNaskh'` and ensure `height: 2.0+` in:

| File | Current | Target |
|------|---------|--------|
| `lib/presentation/surah_screen/surah_screen.dart` (continuous mode) | `Amiri`, `height: 2.2` | `UthmaniTahaNaskh`, `height: 2.2` |
| `lib/presentation/surah_screen/surah_screen.dart` (single-line mode) | `Amiri`, `height: 2.2` | `UthmaniTahaNaskh`, `height: 2.2` |
| `lib/presentation/mushaf_screen/mushaf_screen.dart` (verse text) | `Amiri`, `height: 1.9` | `UthmaniTahaNaskh`, `height: 2.0` |
| `lib/presentation/verse_study/verse_study_screen.dart` | `Uthmani`, `height: 2.0` | `UthmaniTahaNaskh`, `height: 2.0` |

Also update mushaf_screen.dart surah name headers, bismillah, and verse number badges to use `UthmaniTahaNaskh`.

### 2d. User data safety
Bookmarks use `surahId + verseNumber` keys (not text). No Hive migration needed.
